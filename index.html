현재 경로 출력: pwd 파일과 디렉토리 출력: ls 모든 파일과 디렉토리 출력: ls -a 디렉토리 생성: mkdir (경로)(디렉토리명) [예: mkdir 1/2/3 <- 1 아래의 2 아래에 3 디렉토리 생성] 디렉토리 삭제: rmdir (디렉토리명) <- 디렉토리가 비어있어야 함 디렉토리 이동: cd (디렉토리명) (없는)파일 생성, (있는)파일 타임스탬프 갱신: touch (경로)(디렉토리명) 파일 복사: cp (복사할 파일) (저장할 디렉토리 경로) 파일 이름 변경: mv (기존 이름) (변경 후 이름) <- 얘도 당연히 경로 설정 가능! (home에서 sudo snap install tree를 입력한 후) 하위 경로를 트리로 출력: tree vi로 c 파일 만들기. 1. ubuntu에서 vi (파일명).c 입력 2. 왼쪽 아래에 (파일명).c[new]가 표시되어 있을 때, 키보드의 i를 누르면 입력모드로 변환 (i외에도 a,o를 누르면 됨. i: 현재 위치에서 입력, a: 현재 위치 뒤에서 입력, o: 라인을 추가하고 입력) 3. 왼쪽 아래에 --INSERT--가 표시되어 있는 것을 확인한 후, 쓰고싶은 c코드을 입력 4. 코드 작성후 키보드의 ESC를 누르고:wq!를 입력하면 저장하고 종료 (w: 작성write, q: 종료quit) 5. c파일이 생성된 것을 확인 가능
